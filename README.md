# Sales System Project

Projeto standalone do sistema de vendas.

## Rodar localmente

```bash
yarn install
yarn db:setup
yarn dev
```

Aplicação: `http://localhost:3002`

## Observações

- crie `.env` a partir de `.env.example`
- este repositório usa SQLite local por padrão para facilitar execução standalone
- Prisma Client e `.next` são gerados localmente
